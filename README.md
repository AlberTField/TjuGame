Topic：为什么人类喜欢玩游戏？

呼延旭东:

王志勇:
	1.人类游戏是因为人们在游戏中得到反馈/奖励/回报。
	2.人类游戏是因为游戏中存在获得正反馈的机会。
	3.从同一个游戏中，不同的人得到的反馈刺激（满足程度）是不同的，这就是为什么不同人会喜欢不同游戏
	4.现在电子游戏中的情节、竞技技能、故事背景设定等，使得游戏刺激多样化
